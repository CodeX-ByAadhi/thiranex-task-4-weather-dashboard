## Project Overview

SkyCast is a premium weather dashboard application developed for the Thiranex Web Development Internship Task 4. The project demonstrates asynchronous JavaScript, REST API integration, JSON parsing, error handling, and dynamic DOM rendering.

Users can search for any city and view real-time weather details such as temperature, humidity, wind speed, feels-like temperature, weather condition, and a 5-day forecast.

## Key Features

- Search weather by city name
- Fetch real-time data from public REST APIs
- Uses Fetch API with async/await
- Parses and renders JSON data dynamically
- Displays temperature, humidity, wind speed, and weather condition
- Shows 5-day weather forecast
- Handles invalid city names and API errors
- Responsive premium UI design
- Mobile-friendly layout
